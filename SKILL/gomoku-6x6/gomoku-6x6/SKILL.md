# Gomoku-6x6 飞书五子棋

这是一个专为飞书机器人设计的五子棋游戏技能，在 6x6 棋盘上进行对战。

## 游戏说明

### 棋盘规格
- **尺寸**: 6x6 网格
- **坐标**: A1 到 F6（字母表示列，数字表示行）
- **玩家**: 用户执 `X`（先手），AI 执 `O`（后手）

### 胜利条件
- 先在横、竖、斜任意方向连成 5 子者获胜
- 棋盘填满无人获胜则为平局

## 核心功能

### agent.py
主要业务逻辑文件，提供：
- `Gomoku6x6` 类：游戏状态管理
- `draw_board()`: ASCII 棋盘绘制
- `parse_coordinate()`: 坐标解析（如 A1 → (0,0)）
- `format_coordinate()`: 坐标格式化（如 (0,0) → A1）
- `check_win()`: 胜负检测
- `ai_move()`: AI 随机落子
- `get_game_status()`: 获取游戏状态文本

### 与飞书集成
技能设计为可与飞书机器人 API 配合使用：
- 接收用户消息解析为游戏指令
- 返回棋盘图片和/或 ASCII 文本
- 维护每局游戏的状态

## 快速开始

```python
from agent import Gomoku6x6

# 创建游戏
game = Gomoku6x6()

# 玩家落子
game.make_move(0, 0)  # A1

# AI 落子
ai_row, ai_col = game.ai_move()
game.make_move(ai_row, ai_col)

# 获取棋盘显示
board_text = game.draw_board()

# 检查游戏状态
status = game.get_status()
```

## 注意事项

- 6x6 棋盘较小，游戏节奏快
- AI 目前采用随机策略，适合娱乐
- 坐标输入不区分大小写（a1 = A1）
- 非法坐标会提示重新输入
