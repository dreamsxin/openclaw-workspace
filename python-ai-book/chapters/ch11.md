# 第11章 OpenClaw Agent 框架

> OpenClaw 是一个开源的 AI Agent 框架，让你快速构建能「做事」的 AI 助手。

## 11.1 OpenClaw 是什么？

OpenClaw 是一个本地运行的 AI Agent 框架，核心特点：

- **技能系统**：模块化的能力扩展
- **工具集成**：文件、浏览器、终端、API
- **记忆系统**：持久化的上下文管理
- **多渠道**：支持 Telegram、Discord、WhatsApp 等
- **本地优先**：数据不出本机

### 架构概览

```
┌─────────────────────────────────────┐
│           OpenClaw Gateway          │
├─────────────┬───────────────────────┤
│  会话管理    │    技能系统           │
│  (Sessions) │    (Skills)           │
├─────────────┼───────────────────────┤
│  工具层     │    记忆系统           │
│  (Tools)    │    (Memory)           │
├─────────────┴───────────────────────┤
│          模型适配层                  │
│  (Claude / GPT / Ollama / ...)     │
└─────────────────────────────────────┘
```

## 11.2 核心概念

### 技能（Skills）

技能是 OpenClaw 的能力单元，每个技能是一个目录：

```
my-skill/
├── SKILL.md          # 技能说明（AI 读这个来理解如何使用）
├── script.py         # 实现代码
└── config.json       # 配置（可选）
```

SKILL.md 示例：

```markdown
# 天气查询技能

## 描述
查询指定城市的天气信息

## 使用方式
bash weather.sh <城市名>

## 示例
bash weather.sh 北京

## 输出
JSON 格式的天气数据
```

### 工具（Tools）

OpenClaw 内置的工具：

| 工具 | 功能 |
|------|------|
| `exec` | 执行 shell 命令 |
| `read` | 读取文件 |
| `write` | 写入文件 |
| `edit` | 编辑文件 |
| `web_fetch` | 获取网页内容 |
| `browser` | 浏览器自动化 |
| `cron` | 定时任务 |
| `message` | 发送消息 |

### 会话（Sessions）

每个对话是一个会话，有独立的上下文和历史。

## 11.3 创建你的第一个技能

```bash
# 技能目录结构
mkdir -p ~/.openclaw/skills/code-metrics
cd ~/.openclaw/skills/code-metrics
```

创建 SKILL.md：

```markdown
# 代码质量指标

## 描述
分析 Python 代码的质量指标，包括：
- 代码行数
- 函数数量
- 平均函数长度
- 圈复杂度（简化版）

## 使用方式
bash analyze.sh <文件或目录路径>

## 输出格式
JSON 格式：
```json
{
  "files": 10,
  "total_lines": 500,
  "functions": 25,
  "avg_function_length": 15,
  "complexity": "low/medium/high"
}
```
```

创建实现脚本：

```bash
#!/bin/bash
# analyze.sh

TARGET=${1:-.}

python3 - << 'EOF'
import ast
import sys
from pathlib import Path

target = Path(sys.argv[1]) if len(sys.argv) > 1 else Path(".")

files = list(target.rglob("*.py")) if target.is_dir() else [target]
total_lines = 0
total_functions = 0

for f in files:
    try:
        code = f.read_text()
        total_lines += len(code.splitlines())
        tree = ast.parse(code)
        total_functions += sum(1 for node in ast.walk(tree) if isinstance(node, (ast.FunctionDef, ast.AsyncFunctionDef)))
    except Exception:
        pass

avg = total_lines / total_functions if total_functions else 0
complexity = "low" if total_functions < 10 else "medium" if total_functions < 30 else "high"

import json
print(json.dumps({
    "files": len(files),
    "total_lines": total_lines,
    "functions": total_functions,
    "avg_function_length": round(avg, 1),
    "complexity": complexity
}, indent=2))
EOF
```

## 11.5 使用 OpenClaw API

通过 Python 与 OpenClaw 交互：

```python
# code/ch11/openclaw_client.py
import httpx
import json


class OpenClawClient:
    """OpenClaw API 客户端"""

    def __init__(self, base_url: str = "http://localhost:3000"):
        self.base_url = base_url
        self.client = httpx.Client(base_url=base_url)

    def send_message(self, message: str, session: str = "default") -> str:
        """发送消息并获取回复"""
        response = self.client.post("/api/chat", json={
            "message": message,
            "session": session
        })
        return response.json()["response"]

    def list_skills(self) -> list:
        """列出可用技能"""
        response = self.client.get("/api/skills")
        return response.json()["skills"]

    def execute_skill(self, skill: str, params: dict) -> dict:
        """执行技能"""
        response = self.client.post(f"/api/skills/{skill}/execute", json=params)
        return response.json()


# 使用
if __name__ == "__main__":
    client = OpenClawClient()

    # 列出技能
    skills = client.list_skills()
    print("可用技能:", skills)

    # 发送消息
    reply = client.send_message("今天天气怎么样？")
    print("回复:", reply)
```

## 11.6 扩展 OpenClaw

### 自定义工具

```python
# 在技能中注册自定义工具
def register_tools(registry):
    """注册自定义工具"""

    @registry.tool("analyze_code")
    def analyze_code(file_path: str) -> str:
        """分析代码质量"""
        with open(file_path) as f:
            code = f.read()

        tree = ast.parse(code)
        functions = [n for n in ast.walk(tree) if isinstance(n, ast.FunctionDef)]

        return json.dumps({
            "lines": len(code.splitlines()),
            "functions": len(functions),
            "longest_function": max(
                (len(f.body) for f in functions), default=0
            )
        })
```

### Webhook 集成

```python
# 接收 OpenClaw 事件
from fastapi import FastAPI

app = FastAPI()

@app.post("/webhook/openclaw")
async def handle_event(event: dict):
    """处理 OpenClaw 事件"""
    if event["type"] == "message":
        # 处理新消息
        print(f"收到消息: {event['content']}")
    elif event["type"] == "skill_complete":
        # 技能执行完成
        print(f"技能完成: {event['skill']}")
    return {"status": "ok"}
```
