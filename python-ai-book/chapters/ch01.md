# 第1章 环境搭建

> 工欲善其事，必先利其器。本章带你从零搭建一个完整的 Python AI 开发环境。

## 1.1 安装 Python

### 推荐版本

本书代码基于 **Python 3.10+** 编写和测试。推荐使用 3.11 或 3.12。

```bash
# 检查当前版本
python3 --version

# macOS (Homebrew)
brew install python@3.12

# Ubuntu/Debian
sudo apt update && sudo apt install python3.12 python3.12-venv

# Windows — 从 python.org 下载安装包，勾选 "Add to PATH"
```

### 使用 pyenv 管理多版本（推荐）

```bash
# 安装 pyenv
curl https://pyenv.run | bash

# 安装 Python 3.12
pyenv install 3.12.7
pyenv global 3.12.7
```

## 1.2 创建虚拟环境

**永远在虚拟环境中开发。** 这是铁律。

```bash
# 创建项目目录
mkdir my-ai-project && cd my-ai-project

# 创建虚拟环境
python3 -m venv .venv

# 激活
source .venv/bin/activate    # macOS/Linux
# .venv\Scripts\activate     # Windows

# 确认
which python  # 应该指向 .venv 目录
```

## 1.3 安装核心依赖

```bash
# 升级 pip
pip install --upgrade pip

# 安装本书依赖
pip install anthropic openai httpx ollama rich pydantic python-dotenv
```

### requirements.txt

本书每章的代码示例都在 `code/` 目录下，项目根目录有完整的 `requirements.txt`：

```bash
pip install -r requirements.txt
```

## 1.4 配置 API 密钥

### 环境变量方式（推荐）

创建 `.env` 文件（**不要提交到 Git**）：

```bash
# .env
ANTHROPIC_API_KEY=sk-ant-xxxxx
OPENAI_API_KEY=sk-xxxxx
```

在 Python 中加载：

```python
from dotenv import load_dotenv
import os

load_dotenv()

api_key = os.getenv("ANTHROPIC_API_KEY")
print(f"Key loaded: {'Yes' if api_key else 'No'}")
```

### 安全提醒

- **永远不要**把 API 密钥硬编码到代码里
- **永远不要**把 `.env` 文件提交到 Git
- 在 `.gitignore` 中添加 `.env`

```bash
echo ".env" >> .gitignore
```

## 1.6 验证环境

运行以下脚本确认一切就绪：

```python
# code/ch01/verify_env.py
"""环境验证脚本 — 运行此脚本确认开发环境正常"""

import sys
import importlib

def check_package(name):
    try:
        mod = importlib.import_module(name)
        version = getattr(mod, "__version__", "unknown")
        print(f"  ✅ {name} ({version})")
        return True
    except ImportError:
        print(f"  ❌ {name} — 未安装")
        return False

def main():
    print(f"Python {sys.version}\n")

    packages = [
        "anthropic", "openai", "httpx",
        "ollama", "rich", "pydantic",
        "dotenv",  # python-dotenv
    ]

    print("核心依赖检查：")
    results = [check_package(p) for p in packages]

    print(f"\n{'全部通过 ✅' if all(results) else '部分依赖缺失 ❌'}")
    print("请运行: pip install -r requirements.txt")

if __name__ == "__main__":
    main()
```

运行：

```bash
python code/ch01/verify_env.py
```

预期输出：

```
Python 3.12.7

核心依赖检查：
  ✅ anthropic (0.40.0)
  ✅ openai (1.50.0)
  ✅ httpx (0.27.0)
  ✅ ollama (0.4.0)
  ✅ rich (13.9.0)
  ✅ pydantic (2.9.0)
  ✅ dotenv (1.0.0)

全部通过 ✅
```
