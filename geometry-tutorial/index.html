第一章

三角形的性质

三角形是几何学最基本的图形。掌握它的性质，是解决所有几何问题的基石。

📐 三角形内角和 = 180°

任意三角形的三个内角之和恒等于 180°。下面通过可视化来理解为什么：

1过顶点 C 作直线 DE ∥ AB（平行于底边）
2由内错角相等：∠DCA = ∠A，∠ECB = ∠B
3∠DCA + ∠ACB + ∠ECB = 180°（平角）
4因此 ∠A + ∠B + ∠C = 180° ∎

🔄 全等三角形判定

两个三角形全等，意味着它们形状和大小完全相同。判定方法有五种：

  • SSS — 三边对应相等
  • SAS — 两边及夹角对应相等
  • ASA — 两角及夹边对应相等
  • AAS — 两角及一对等角的对边相等
  • HL — 直角三角形：斜边 + 一直角边相等

SSS：三边对应相等 → 两个三角形全等

第二章

勾股定理

直角三角形中最优美的关系：两条直角边的平方和等于斜边的平方。

📏 a² + b² = c²

在直角三角形中，设两直角边为 a、b，斜边为 c：

a² + b² = c²

赵爽弦图证明（面积法）：

1大正方形边长 = a + b，面积 = (a+b)²
2内部小正方形面积 = c²
3四个直角三角形面积 = 4 × ½ab = 2ab
4(a+b)² = c² + 2ab
5a² + 2ab + b² = c² + 2ab
6∴ a² + b² = c² ∎

第三章

圆的性质

圆是初中几何的重点和难点。掌握圆心角、圆周角、切线等核心定理至关重要。

⭕ 圆心角与圆周角

圆心角：顶点在圆心的角。圆周角：顶点在圆上的角。

核心定理：圆周角 = ½ × 同弧上的圆心角

📏 切线的性质

直线与圆只有一个公共点时，称为圆的切线。

  • 切线⊥半径：切线与过切点的半径垂直
  • 切线长相等：从圆外一点引两条切线，切线长相等
第四章

面积公式

掌握常见图形的面积计算，是解题的基本功。

🔺 三角形

S = ½ × 底 × 高

▱ 平行四边形

S = 底 × 高

⬡ 梯形

S = ½ × (上底 + 下底) × 高

⭕ 圆

S = πr²
C = 2πr
