## 一、世界观与故事背景

### 1.1 太初纪元

> *"太初有道，道生一，一生二，二生三，三生万物。"*

洪荒之初，混沌一片，无天无地，无阴无阳。  
**鸿钧老祖**盘坐紫霄宫，以大道之力化育三清，是为天地间第一位存在。

此后，**盘古**持斧劈开混沌，轻清之气上升为天，重浊之气下沉为地，以肉身化万物——眼化日月，血化江河，骨化山岳。天地成形，造化初开。

**女娲**以黄土抟造人族，炼五色神石补天，自此三界有序，众生繁衍。

然而随着时代更迭，封神之战爆发，三界秩序大乱。阴阳失衡，五行错位，上古神明或陨落、或沉眠，洪荒秩序濒临崩塌。

### 1.2 执命人的使命

玩家扮演来自洪荒长河的**执命人**——一位能够召唤上古神明灵魂、操控命运轨迹的神秘旅者。

执命人可在六大称号中选择身份：

| 称号 | 描述 |
|------|------|
| 无名执命人 | 来自洪荒长河的旅人 |
| 观星执命人 | 能窥见命运轨迹 |
| 守序执命人 | 维护三界秩序的使者 |
| 混沌执命人 | 游走于规则之外 |
| 仙道执命人 | 追求大道的修行者 |
| 征伐执命人 | 以战止战的斗神后裔 |

执命人的目标是：**召唤并修炼上古神明，率领神明队伍出征应战，重建三界秩序。**

### 1.4 五大战场（混沌遗迹）

当前版本开放**混沌遗迹**作为主要战斗场景，分多层递进，强度依层递增。敌方由系统随机生成的神明阵营构成，玩家需率五人队伍出征应战。

## 三、战斗系统

### 3.1 基础战斗框架

本作采用**回合制战棋战斗**，双方各出5名神明（玩家最多选5人，敌方由系统配置）。

#### 行动顺序

所有上场单位按**速度（SPD）从高到低**排列行动队列，速度越高越早行动。每回合结束后重新排列存活单位队列。

```
行动队列示例：
金乌(95) → 陆压(88) → 嫦娥(85) → 帝俊(82) → 东皇(78) → ...
```

#### 回合流程

```
每回合开始
    ↓
  结算回合开始状态效果（灼烧扣血、回血恢复）
    ↓
  当前行动者为玩家 → 玩家选择普攻/技能/目标
  当前行动者为敌方 → AI自动行动
    ↓
  执行行动 → 触发属性克制 → 结算伤害 → 应用状态效果
    ↓
  检查死亡 → 更新行动队列
    ↓
  轮至下一行动者，如此循环
```

### 3.2 属性克制关系

#### 第一层：无极超然

| 攻击方 | 防御方 | 效果 |
|--------|--------|------|
| 无极 | 任意 | 混沌无视：必然命中，无视一切克制关系，伤害恒定 |

#### 第二层：阴阳对立

| 攻击方 | 防御方 | 伤害倍率 | 附加效果（触发率） |
|--------|--------|----------|---------------------|
| 阴 | 阳 | ×1.3 | 概率沉默目标一回合（40%） |
| 阳 | 阴 | ×1.3 | 概率眩晕目标一回合（40%） |
| 阴/阳 | 五行 | ×1.15 | 额外造成目标最大生命值5%的真实伤害（必触发） |
| 五行 | 阴/阳 | ×0.85 | 伤害降低15%，无法暴击 |

#### 第三层：五行循环

循环顺序：**金 → 木 → 土 → 水 → 火 → 金**

| 克制关系 | 伤害倍率 | 附加效果 |
|----------|----------|----------|
| 金克木 | ×1.25 | 破甲驱散，移除目标一个增益 |
| 木克土 | ×1.25 | 枝蔓缠绕，移除目标一个增益 |
| 土克水 | ×1.25 | 大地吞噬，移除目标一个增益 |
| 水克火 | ×1.25 | 冰封驱散，移除目标一个增益 |
| 火克金 | ×1.25 | 炎焰灼烧，移除目标一个增益 |
| 被克关系 | ×0.8 | 属性不利 |
| 同属性/无克制 | ×1.0 | — |

### 3.3 伤害计算公式

```
基础伤害 = 技能倍率 × 攻击力（普攻时 = 攻击力）
实际伤害 = max(1,  基础伤害 − 防御力 × 0.5)
           × 属性克制倍率
           × 暴击修正（暴击时 × 暴击伤害倍率）

若技能具有"穿甲破防"效果：
  实际防御 = 防御力 × (1 − 穿甲率)
```

**真实伤害**无视防御，直接扣除生命值。  
**斩杀阈值**：部分技能（如金神执钺）在目标血量低于30%时直接将其HP归零。

### 3.4 技能系统

每位神明拥有 **2个主动技能** 和 **1个被动技能**。

#### 怒气机制（Rage）

- 玩家方英雄拥有怒气槽，上限为5点
- 普攻命中/受到伤害时积累怒气
- 技能释放需消耗怒气（技能Cost 2~5不等）
