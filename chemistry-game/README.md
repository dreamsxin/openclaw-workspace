# 🧪 化学周期表探索游戏

一款面向初中生的交互式化学学习游戏，帮助学生记忆元素周期表和化学表达式。

## 🚀 功能特性

- **交互式周期表** - 点击任意元素查看详细信息
- **颜色分类** - 8 种元素类别用不同颜色区分
- **小测验模式** - 随机出题，测试学习成果
- **🃏 卡片记忆游戏** - 翻牌匹配，学习电子结构和化合价
- **📖 学习模式** - 浏览所有元素详细卡片
- **⚛️ 电子壳层可视化** - 直观展示电子排布
- **得分系统** - 记录学习进度
- **响应式设计** - 支持电脑和平板

## 📦 安装运行

```bash
# 进入项目目录
cd chemistry-game

# 安装依赖
npm install

# 启动游戏
npm start

# 开发模式（自动重载）
npm run dev
```

## 🎮 使用方法

1. 打开浏览器访问 `http://localhost:3000`
2. **查看元素** - 点击周期表中的任意元素方块
3. **开始测验** - 点击"开始测验"按钮或按空格键
4. **回答问题** - 输入答案后按回车或点击提交

## 📚 学习模式

- **元素符号记忆** - 记住每个元素的化学符号
- **原子序数学习** - 了解元素的原子序数
- **类别识别** - 通过颜色识别元素类别（碱金属、卤素、稀有气体等）

## 🛠️ 技术栈

- **后端**: Node.js + Express
- **前端**: 原生 HTML/CSS/JavaScript
- **模板**: EJS
- **数据**: JSON 格式存储元素信息
